# Students Performance Report

The purpose of this report is to explain how to increase students performance acording to their final exam score

This report discusses:
1) What increase students performance like "study habits"
2) What decrease students performance like "distraction" and how we can make students more interested in their education
3) What effect has the "health" of the student to their perfomance
